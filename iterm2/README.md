# iTerm2 Setup

## Font
- Font: **JetBrainsMono Nerd Font**
- Size: 14
- Install via: `brew install --cask font-jetbrains-mono-nerd-font`

## Color Scheme
- Import **Gruvbox Dark** from: https://github.com/morhetz/gruvbox-contrib/tree/master/iterm2
- Profiles > Colors > Color Presets > Import

## Settings
- Profiles > Terminal > Enable "Applications in terminal may access clipboard"
- Profiles > Keys > General > Left Option key: Esc+
